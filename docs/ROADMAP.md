# Roadmap

## Phase 1: Source Extraction

- Convert `source.pdf` to machine-readable text.
- Clean OCR artifacts and normalize formatting.
- Tag key statements by topic.

## Phase 2: Structure

- Define manifesto chapter outline.
- Map extracted statements to each chapter.
- Identify missing requirements and open questions.

## Phase 3: Drafting

- Produce first full draft (`v0.1`).
- Tighten language around engineering constraints.
- Add rationale for major design choices.

## Phase 4: Validation

- Run consistency review across requirements.
- Check feasibility of manufacturing and integration claims.
- Finalize `v1.0` publication draft.
